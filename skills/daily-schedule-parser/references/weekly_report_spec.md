# 周报生成规范

## 触发条件

- 用户主动请求："帮我生成本周报告"、"周报"等
- 每周六/日 Agent 自动提醒用户是否生成周报

## 数据来源

读取过去 7 天（周一到周日）的 CSV 记录，按 date 字段过滤。

## 报告结构

### 1. 时间分配总览

- 本周总记录时长（小时）
- 主分类饼图数据（category → total duration_min）
- 子分类明细（每个主分类下 top 3 子分类）

示例输出：

```
📊 本周时间分配（共记录 52.5 小时）
- 工作：22h（42%）→ 开发 8h / 规划 6h / 沟通 5h
- 生活：14h（27%）→ 用餐 5h / 休闲 4h / 家务 3h
- 通勤：6h（11%）
- 健康：5.5h（10%）→ 睡眠 3h / 运动 2h
- 社交：5h（10%）
```

### 2. 高频事件 Top 10

- 按 tags 或 sub_category 出现频次排序
- 每个高频事件显示：出现次数 + 最近一次的 content + 相关的 reflection 摘要
- 若 reflection 中有进展追踪（如"还在推进"、"已完成"），突出显示

示例输出：

```
🔁 高频事件

1. 年度规划（出现 4 次）
   最近：2/25 撰写工作年度计划，完成框架
   感想："需要在下周前定稿"

2. 职业引荐（出现 2 次）
   最近：2/25 帮表妹与陈婉对接财务专员职位
   感想："希望他们能沟通顺利"
```

### 3. 情绪与状态洞察

- 从 reflection 字段中提取关键情绪词
- 统计正面/中性/负面情绪比例
- 列出本周出现的主要关注点或待办事项（从 reflection 中提取"需要"、"计划"、"希望"等关键词）

示例输出：

```
💭 本周状态洞察
- 整体情绪：积极为主（正面 65% / 中性 28% / 负面 7%）
- 本周关注点：
  · 年度规划定稿
  · 表妹职业引荐进展
  · AI 工具推广分享
```

## generate_weekly_report.py 调用规范

```python
python scripts/generate_weekly_report.py \
  --data-dir ~/openclaw-data/schedules/ \
  --username john_doe \
  --week-start 2026-02-23 \
  --output ~/openclaw-data/reports/weekly_2026-02-23.md
```

输出为 Markdown 格式，包含图表数据（用 ASCII 或 Mermaid 饼图表示）。
