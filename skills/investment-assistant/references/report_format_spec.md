## 评估记录 CSV 字段说明

写入 `evaluations_YYYY-MM.csv` 的字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| date | 评估日期 | 2026-02-26 |
| symbol | 股票代码 | TSLA |
| market | 市场 | US |
| direction | 计划方向 | long |
| plan_id | 关联计划 ID | plan_001 |
| target_price | 目标价格 | 450.00 |
| current_price | 当前收盘价 | 408.50 |
| price_in_range | 是否在目标区间 | true |
| technical_score | 技术面评分 | 62 |
| news_score | 消息面评分 | 55 |
| fundamentals_score | 基本面评分 | 68 |
| verdict | 最终决策 | wait |
| confidence | 信心度 | 65 |
| reason | 简要原因 | 等待MACD金叉确认 |

## 风格指南

- 使用 emoji 增强可读性，但不要过度
- 数字精确到小数点后两位（价格）或整数（评分）
- 中文为主，专业术语保留英文（如 MACD, RSI, PE）
- 每次推送控制在 Discord 单条消息合理长度内（约 2000 字符）
- 重要信息放在开头，详细分析可以展开查看
